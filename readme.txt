=== Detalhes do Projeto ===


== Tecnologia utilizada ==
Angular 4


== O que foi realizado ==

1) Login com Google ou Facebook;
2) Lista de produtos com foto;
3) Página com detalhes do produto;
4) Pequeno sistema de comentários na página de detalhes;
5) Logout


== Adições ==

O sistema se comunica com um sistema externo para pegar os dados via API. Da mesma forma lê e salva os comentários.
A API foi desenvolvida com C#, utilizando bando de dados MariaDB (MySQL).
